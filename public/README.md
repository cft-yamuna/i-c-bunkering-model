# Asset Requirements

Place the following files in this `public` directory:

## Images
- `bg.png` - Background image for start screen
- `start.png` - Start button image
- `b1.png` - Button 1 normal state
- `b1activated.png` - Button 1 activated state
- `b2.png` - Button 2 normal state
- `b2activated.png` - Button 2 activated state
- `b3.png` - Button 3 normal state
- `b3activated.png` - Button 3 activated state

## Videos
- `bg.mp4` - Background video for button selection screen
- `1.mp4` - Video for button 1
- `2.mp4` - Video for button 2
- `3.mp4` - Video for button 3
